# dyrman.com

personal homepage built with pure F#

## structure

- `src/Notes` — static site generator (markdown to HTML)
- `notes/` — posts (long-form thoughts, essays, etc.)
- `stream/` — short-form thoughts (tweets, microblogs, etc.)
- `library/` — reading list (books, articles, podcasts, etc.)
- `articles/` — curated article recommendations
- `homepage/` — generated output
